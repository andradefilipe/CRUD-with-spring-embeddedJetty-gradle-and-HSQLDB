Spring Embedded Database
===============================
Template for a Spring 4 MVC + Embedded Database examples, using HSQLDB.

###1. Technologies used
* Gradle
* Maven 3.0
* Spring 4.1.6.RELEASE
* HSQLDB 2.3.2

###2. To Run this project locally
```shell
$ gradle jettyRun
```
Access ```http://localhost:8080/spring-mvc-db/```
